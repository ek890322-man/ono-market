# ONO MARKET 회원 시스템 추가판

실행: Windows에서 start.bat 더블클릭
또는:
pip install -r requirements.txt
python app.py

쇼핑몰: http://127.0.0.1:5000
관리자: http://127.0.0.1:5000/admin
기본 관리자 비밀번호: change-me

추가 기능:
- 회원가입 / 로그인 / 로그아웃
- 비밀번호 해시 저장
- 마이페이지
- 회원정보/기본 배송지 수정
- 비밀번호 변경
- 회원 탈퇴
- 회원 주문내역
- 로그인 주문과 회원 연결
- 관리자 회원 목록
- 회원별 주문 건수 / 누적 구매액
- 회원 정상/정지/탈퇴 상태 관리

실서비스 전 SECRET_KEY와 ADMIN_PASSWORD를 환경변수로 반드시 변경하세요.
카드 결제는 아직 미연동입니다.


관리자 페이지에 쇼핑몰 디자인 관리 기능 추가: 쇼핑몰명, 메인 제목/설명, 버튼색, 배경색, 배너 그라데이션 색상, 실시간 미리보기 및 저장 즉시 반영.
